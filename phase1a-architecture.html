二、功能模块

📋 项目管理

  • 新建/编辑/删除项目
  • 多房间支持 (暗室/控制室/功放室/屏蔽室)
  • 房间尺寸 & 照明配置
  • 线路管理 (新增/编辑/删除线路)

🔌 线路编辑器

  • 选择滤波器 → 自动推荐线缆规格
  • CableSegment 线段编辑 (主干/分支/并联/拖链)
  • 末端设备配置 (插座类型 & 数量)
  • 电缆长度输入 (支持人工填写或坐标计算)

📊 BOM & 成本

  • 实时物料清单
  • 按类别分组 (滤波器/电缆/插座/附件)
  • 单价 × 数量 = 小计 → 总计
  • 导出 Excel (.xlsx)

🔍 GB 安全校验

  • 载流量校验 (GB/T 16895)
  • 电压降校验 (GB 50054 ≤5%)
  • 热稳定 & 机械强度校验
  • 生成独立合规报告 (绿色通过/红色警告)

📦 滤波器库管理 /admin

  • 列表 + 搜索 + 筛选
  • 单条编辑 (型号/参数/尺寸/价格)
  • 批量导入 Excel
  • 批量导出 (备份/分享)

📐 线缆规格管理 /admin

  • 载流量表可视化编辑
  • 新增电缆型号
  • 校正系数配置 (温度/并列/敷设方式)

📜 国标数据管理 /admin

  • 载流量表查看/更新
  • 安全规则配置
  • 版本管理 (追踪更新历史)
  • 支持定期自动 & 手动更新

💰 价格管理 /admin

  • 批量上传新价格表 (Excel)
  • 价格变更历史
  • 供应商价格版本追踪

三、数据模型

Project ├─ id, name, created_at └─ rooms[] └─ Room ├─ type: 暗室 | 控制室 | 功放室 | 屏蔽室 ├─ dimensions: { length, width, height } ├─ lighting: { model, count, circuits } └─ circuits[] └─ Circuit ← 核心实体 ├─ filter_id ─────→ FilterLibrary ├─ purpose: 照明 | 设备 | EUT | 转台 | 预留 ├─ voltage_phases: 单相 | 三相 | 直流 └─ segments[] ← 线段列表 └─ CableSegment ├─ type: trunk | branch | parallel | dragchain ├─ cable_spec_id ──→ CableSpecs ├─ length_m ├─ parallel_count: 1 (并联根数) ├─ parent_segment_id: null (用于分支拓扑) ├─ from_location: string (起点描述) └─ to_location: string (终点描述) └─ devices[] └─ Device ├─ type: 插座 | 连接器 | 灯 | 设备端子 ├─ rating: { V, A } (额定值) └─ quantity: N

拓扑示例

照明线路 (滤波器 DL-2X32C55) trunk: 滤波器 → 灯区接线盒 (YZW 3×6mm², 18m) ├─ branch: 接线盒 → 灯×4 (YZW 3×2.5mm², 6m/根) └─ branch: 接线盒 → 灯×5 (YZW 3×2.5mm², 8m/根) 转台线路 (滤波器 DL-4X63C55) ← 拖链场景 trunk: 滤波器 → 拖链入口 (YZW 4×16mm², 22m) └─ parallel ×3: 拖链内 (ZC-TRVV 4×16mm², 38m/根, 柔性电缆) └─ 转台端子 (380V, 63A, 工业连接器 ×1) 天线塔插座线路 (滤波器 DL-2X32C55) trunk: 滤波器 → 开口板接线盒 (YZW 3×6mm², 25m) ├─ branch: 10A 插座 ×2 (YZW 3×2.5mm², 3m) └─ branch: 16A 插座 ×2 (YZW 3×4mm², 3m)

五、GB 安全校验模块

检查项依据标准规则输出
载流量 GB/T 16895 Iz · Kt · Kg ≥ 1.25 × Ib
(载流量 × 温度系数 × 并列系数 ≥ 回路电流 × 1.25)		 ✓ 通过 / ✗ 建议增大截面
电压降 GB 50054 §3.4 ΔU% = (√3 × I × L × cosφ) / (γ × S × U) × 100% ≤ 5% ✓ 通过 / ✗ 压降超标
热稳定 GB 50217 S ≥ (Ik × √t) / K ✓ 满足 / ✗ 不满足
机械强度 GB 50054 动力线 ≥ 2.5mm², 照明 ≥ 1.5mm² ✓ 满足 / ✗ 不满足
电压等级 GB 50052 滤波器额定电压 ≥ 系统标称电压 ✓ 匹配 / ✗ 不匹配

九、审查决策记录

#决策结论
D1范围拆分Phase 1a (表单版核心引擎) → Phase 1b (可视化编辑器)
D2项目结构Vite monorepo，前后端同仓库

D4GB 国标核心规则内置 (载流量表+电压降)，配置表形式，安全项独立模块

D6数据管理全部参考数据通过 Web 界面管理，支持单条编辑和批量导入
