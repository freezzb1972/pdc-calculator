一、Excel 导入

命令行

导入npm run import-excel <文件.xlsx> [--type=filters|cables|all] [--sheet=SheetName] [--dry-run]
库exceljs v4.4.0
匹配列头模糊匹配（大小写不敏感、子串包含）

1 必填

型号 / 名称 / model必填string (唯一)—如 DL-2X16C55-JCL-H
厂商 / 厂家 / 制造商 / 品牌 / manufacturer可选string坚力
电压 / 额定电压 / voltage可选number空额定电压 V
电流 / 额定电流 / current可选number空额定电流 A
相数 / 相 / phase可选string三相单相 / 三相
线数 / 线 / wire / 芯数可选integer4
单价 / 价格 / price可选number0
类别 / 类型 / category / 用途可选string暗室设备照明 / 暗室设备 / EUT / 转台
示例 Excel 行
DL-2X16C55-JCL-H | 坚力 | 250 | 16 | 单相 | 2 | 909.87 | 照明

Sheet 2 — 电缆规格

1 必填
Excel 列头关键字必填类型默认值说明
型号 / 名称 / model必填string (唯一)—如 YZW 3×1.5mm²

四、JSON API 批量导入 — 价格更新

POST /api/prices/import

仅更新，不创建新记录
字段必填类型说明
type必填"filter" | "cable"更新滤波器的价格还是电缆的价格
items[].id必填integerfilters.id 或 cable_specs.id
items[].unit_price必填number新单价
JSON 请求体
{"type": "filter", "items": [{"id": 1, "unit_price": 950.00}, {"id": 2, "unit_price": 1200.00}]}
旧价格自动写入 price_history 表，source 标记为 "batch_import"。
